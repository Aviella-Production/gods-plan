## 🔍 Downloading Assets

### Requirements
- [Node.js](https://nodejs.org/en/)
- An internet connection.

### Gaining Permission
We expect you to have a permission to access the asset folder. If no please contact [RJ-045](https://github.com/RJ-045).

### Downloading
Run command below:
```sh
$ npm install
$ npm run build
$ npm start
```
